## 📊 巨量广告平台概述

### 支持的广告平台
- **抖音**: 短视频广告（信息流、开屏、品牌）

- **穿山甲**: 聚合广告平台

### 广告类型
1. **信息流广告**: 用户浏览内容时穿插广告
2. **开屏广告**: 打开 APP 时全屏广告
3. **品牌广告**: 品牌曝光和认知
4. **效果广告**: 转化（下载、注册、付费）

## 🎯 核心准备工作

### 1. 账户准备（1-2 天）

#### 资质准备
- [ ] **营业执照**: 企业注册证明
- [ ] **ICP 备案**: 网站备案（如果投放网页广告）
- [ ] **税务登记**: 税号（用于发票）
- [ ] **银行账户**: 对公账户
- [ ] **法人身份证**: 正反面照片
- [ ] **行业资质**: 特殊行业需要许可证（如医疗、金融、游戏）

#### 开户流程
1. 登录巨量引擎官网（https://www.oceanengine.com）
2. 注册广告主账户
3. 提交资质审核
4. 充值广告费用（最低充值：1 万起）
5. 等待审核（1-3 个工作日）

### 2. 产品准备（1 周）

#### 产品核心功能
- [ ] **对话功能**: 多轮对话、长上下文
- [ ] **用户体验**: 界面简洁、响应迅速
- [ ] **转化漏斗**: 广告点击 → 下载/注册 → 体验 → 付费
- [ ] **埋点系统**: 统计转化数据

#### 优化建议
- **落地页**: 设计专门的广告落地页
- **下载引导**: 一键下载，减少步骤
- **首次体验**: 提供免费对话次数
- **付费引导**: 明确付费标准和收益

### 3. 创意准备（3-5 天）

#### 视频创意（抖音）
- [ ] **15 秒短视频**: 快速吸引用户
- [ ] **30 秒深度视频**: 详细展示功能
- [ ] **口播视频**: 主播介绍
- [ ] **场景化视频**: 对话场景演示

#### 图文创意（头条、西瓜视频）
- [ ] **横幅广告**: 3:1 比例
- [ ] **信息流广告**: 1:1 或 4:3 比例
- [ ] **Banner 广告**: 640x100 像素

#### 创意方向
1. **痛点切入**: "写论文没思路？用 AI 对话帮帮你"
2. **场景演示**: 展示多轮对话
3. **用户证言**: "100 万用户的对话神器"
4. **免费试用**: "新用户 10 次免费对话"

### 4. 定向准备（1-2 天）

#### 人群定向
- **年龄**: 18-45 岁（重点 22-35 岁）
- **性别**: 男女性别（根据产品调整）
- **地域**: 一线城市（北上广深）或全国
- **兴趣**: 科技、学习、写作、AI
- **设备**: iOS 优先（付费能力高）

#### 行为定向
- **搜索行为**: "AI 对话"、"大模型"、"智能助手"
- **浏览行为**: 科技类、学习类内容
- **安装行为**: 已安装类似 APP 的用户

### 5. 预算准备（灵活）

#### 预算建议
| 时期 | 日预算 | 目标 |
|------|--------|------|
| **测试期** | ¥1,000-2,000/天 | 测试创意和定向 |
| **优化期** | ¥5,000-10,000/天 | 优化投放 |
| **扩张期** | ¥10,000-50,000/天 | 大规模投放 |

#### 费用构成
- **广告费用**: CPC（按点击付费）、CPM（按展示付费）、CPI（按下载付费）
- **素材费用**: 视频制作（2,000-5,000/个）
- **运营费用**: 广告优化（5,000-10,000/月）
- **其他**: 数据分析、A/B 测试

4. **成本**: CPC、CPM、CPI

## 🚀 投放流程

### Day 1: 账户开户
- 提交资质申请
- 等待审核通过

### Day 2-3: 创意准备
- 制作视频广告
- 制作图文广告
- 准备落地页

### Day 4: 测试投放
- 设置小预算（1,000/天）
- 测试 3-5 个创意
- 测试 3-5 个定向

### Day 5-7: 优化调整
- 暂停效果差的创意
- 加大效果好的创意预算
- 优化定向人群

### Week 2-4: 规模化投放
- 扩大预算（5,000-10,000/天）
- 新增 3-5 个创意
- 新增 3-5 个定向

### Month 2-3: 持续优化
- A/B 测试对比
- 优化转化漏斗
- 提高用户留存
