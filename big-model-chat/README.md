## 🎯 项目简介

大模型对话系统（Big Model Chat）是一个完整的对话管理系统，支持多模型、长上下文、工具调用，为自研大模型提供测试环境。

---

## 🚀 快速开始

### 方案 A: 使用 OpenClaw Gateway（推荐）

**优势**:
- ✅ 无需部署，直接对接模型 API
- ✅ 快速验证功能
- ✅ 无服务器成本

**步骤**:
1. 在 LemClaw 添加对话页面
2. 在 LemClaw 配置模型 API Key
3. 开始测试

---

## 🚀 方案 B: 自建完整后端（推荐用于长期）

**优势**:
- ✅ 完全控制和定制
- ✅ 数据完全本地
- ✅ 支持离线运行
- ✅ 可以深度优化

**技术栈**:
- 后端: Python + FastAPI + SQLAlchemy + MySQL + Redis + Celery
- 前端: React + TypeScript + Tailwind CSS
- 数据库: MySQL（用户） + Redis（会话）
- 队列: Celery（任务）
- 缓存: Redis（热点数据）

## 🛠️ 问题排查

**常见问题**:
1. 端口冲突
2. 数据库连接失败
3. 模型 API 调用限制
4. WebSocket 断线
5. 长上下文管理

**解决方法**:
1. 查看日志 `tail -f logs/app.log`
2. 检查配置 `.env`
3. 使用监控脚本 `./monitor.sh`
