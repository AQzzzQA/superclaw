# DSP国内全媒体广告平台 - 技术架构文档

## 项目概述

DSP（Demand Side Platform）国内全媒体广告平台是一个面向广告主的全渠道广告投放管理平台，支持多个媒体渠道的广告投放、数据监控、效果分析和智能优化。

## 技术栈

### 后端技术栈
- **Web框架**: FastAPI 0.104+
- **ORM**: SQLAlchemy 2.0+
- **数据库**: MySQL 8.0+
- **缓存**: Redis 7.0+
- **任务队列**: Celery 5.3+
- **API风格**: RESTful API + WebSocket
- **认证**: JWT + OAuth2.0

### 前端技术栈
- **框架**: React 18+
- **UI组件**: Ant Design 5+
- **状态管理**: Redux Toolkit
- **构建工具**: Vite

### 部署技术栈
- **容器**: Docker + Docker Compose
- **反向代理**: Nginx
- **进程管理**: Supervisor

## 技术亮点

1. **高性能**: FastAPI异步框架 + Redis缓存 + 数据库优化
2. **高可用**: 集群部署 + 主从复制 + 自动故障转移
3. **实时性**: WebSocket推送 + 异步任务队列
4. **安全性**: JWT认证 + OAuth2.0 + 数据加密
5. **可扩展**: 微服务架构 + 插件化设计
6. **易维护**: 完整的日志 + 监控 + 文档

## 技术选型理由

| 技术组件 | 选型理由 |
|---------|---------|
| FastAPI | 高性能异步框架，自动API文档，类型安全 |
| SQLAlchemy | 成熟的ORM，支持异步操作，数据库迁移友好 |
| MySQL 8.0 | 成熟稳定，性能优秀，支持JSON字段 |
| Redis 7.0 | 高性能缓存，支持多种数据结构，持久化 |
| Celery | 成熟的任务队列，支持多种消息代理，定时任务 |
| React + Ant Design | 组件化开发，丰富的UI组件，生态完善 |
