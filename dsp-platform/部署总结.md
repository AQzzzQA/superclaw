## 🔧 服务管理

### 启动所有服务
```bash
cd /root/.openclaw/workspace/dsp-platform
docker-compose up -d
```

### 停止所有服务
```bash
cd /root/.openclaw/workspace/dsp-platform
docker-compose down
```

### 查看服务状态
```bash
docker-compose ps
```

### 查看服务日志
```bash
docker-compose logs -f [service-name]
```

### 重启单个服务
```bash
docker-compose restart [service-name]
```
