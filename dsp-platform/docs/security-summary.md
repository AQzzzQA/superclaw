#### 章节3：核心安全模块

### 3. 数据脱敏工具 ✅

**文件**: `/root/.openclaw/workspace/dsp-platform/backend/app/utils/data_masker.py`

**功能**：
- 邮箱地址脱敏
- 手机号码脱敏
- 身份证号脱敏
- 密码脱敏
- Token脱敏
- 银行卡号脱敏
- 字典自动脱敏
- 日志消息自动脱敏
- JSON脱敏

**特性**：
- 自动识别敏感数据类型
- 支持自定义脱敏规则
- 支持嵌套字典和列表
- 正则表达式匹配

**使用示例**：
```python
from app.utils.data_masker import masker

# 脱敏字典
user_data = {"email": "admin@example.com", "phone": "13812345678"}
masked = masker.mask_dict(user_data)

# 脱敏日志
log_msg = "User admin@example.com logged in"
masked_log = masker.mask_log_message(log_msg)
```
