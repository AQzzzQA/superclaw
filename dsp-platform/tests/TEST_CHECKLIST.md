## 测试执行计划

### 本地开发

```bash
# 运行单元测试
pytest tests/unit -v

# 运行带覆盖率的测试
pytest --cov=app tests/unit

# 运行集成测试
pytest tests/integration -v

# 运行E2E测试
pytest tests/e2e -v --browser=chromium
```

### CI/CD 流水线

```yaml
test:
  script:
    - pytest tests/unit -v --cov=app --cov-report=xml
    - pytest tests/integration -v
    - coverage_report
  only:
    - merge_requests
    - main
```

### 发布前检查

```bash
# 运行所有测试
pytest tests/ -v

# 生成覆盖率报告
pytest --cov=app --cov-report=html
open htmlcov/index.html
```
