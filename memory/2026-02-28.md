## 技术难点

### 1. 浏览器自动化
- agent-browser 需要 gateway token 才能在本地使用，无法远程访问
- 服务器无图形浏览器，无法可视化调试

### 2. React 渲染问题
- 路由配置复杂，容易出错
- HMR 热更新问题导致白屏

### 3. 网络访问
- 外网访问需要开放安全组和端口
- 本地回环地址 (127.0.0.1) 无法从外网访问

### 4. 巨量 API 对接
- API 签名算法复杂
- 需要正确的 app_id 和 app_secret
- Token 刷新机制需要手动触发

### 5. 前后联调
- CORS 配置
- Token 传递机制
- 错误处理
- 日志记录
