### 1. Backend 容器健康检查失败
- **时间**: 2026-03-16 09:10
- **问题**: Backend 容器显示 unhealthy
- **原因**: 健康检查端点路径错误（/api/v1/health vs /api/v1/system/health）
- **严重程度**: 🔴 高
- **影响**: 健康监控失效

### 2. 项目目录混淆
- **时间**: 2026-03-16 09:20
- **问题**: dsp-platform-production 目录为空，但容器使用此目录
- **原因**: Git 操作导致目录清空
- **严重程度**: 🔴 高
- **影响**: 容器重启后无法找到代码

### 3. Celery 服务连接失败
- **时间**: 2026-03-16 09:30
- **问题**: Celery Worker/Beat/Flower 无法连接到 Redis
- **原因**: Celery 环境变量缺少 Redis 密码
- **严重程度**: 🔴 高
- **影响**: 异步任务处理失效

### 4. Celery Beat 配置错误
- **时间**: 2026-03-16 09:40
- **问题**: Celery Beat 使用了不存在的 django_celery_beat 调度器
- **原因**: 依赖模块未安装
- **严重程度**: 🟡 中
- **影响**: 定时任务无法启动

### 5. Flower 容器启动失败
- **时间**: 2026-03-16 09:45
- **问题**: Flower 无法找到 worker 模块
- **原因**: 缺少卷挂载和 PYTHONPATH
- **严重程度**: 🟡 中
- **影响**: Celery 监控界面不可用
