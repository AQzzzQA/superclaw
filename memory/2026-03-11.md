### Nginx 配置

创建了 3 个反向代理规则：

1. **`/gateway`** - OpenClaw 网关
   ```nginx
   location ^~ /gateway {
       proxy_pass http://127.0.0.1:18789;
   }
   ```

2. **`/admin/`** - 前端界面
   ```nginx
   location ^~ /admin/ {
       proxy_pass http://127.0.0.1:3000/;
   }
   ```

3. **`/api/`** - 后端 API
   ```nginx
   location ^~ /api/ {
       proxy_pass http://127.0.0.1:3001/;
   }
   ```

4. **`/assets/`** - 静态资源（新增）
   ```nginx
   location ^~ /assets/ {
       proxy_pass http://127.0.0.1:3000;
   }
   ```

### 问题修复

#### 问题：白屏
**原因**：静态资源（`/assets/`）没有代理到后端

**解决**：创建 `permissions-static.conf` 配置文件，代理 `/assets/` 路径到前端容器

```

Nginx 反向代理

```
