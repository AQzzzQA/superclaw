### 技能特性
**Page Agent** 是阿里巴巴开源的网页智能体：

#### 核心能力
- 🎯 网页自动化：用自然语言控制网页界面
- 🌐 纯 JavaScript：无需浏览器扩展、Python 或无头浏览器
- 🧠 灵活的 LLM 支持：通义千问、OpenAI、Claude 等
- 🎨 精美 UI：支持人机协作模式

#### 使用场景
1. **SaaS AI 助手** - 为产品添加 AI 助手，无需重写后端
2. **智能表单填写** - 将 20 步工作流简化为一句话
3. **无障碍访问** - 语音命令、屏幕阅读器
4. **跨页面任务** - 配合 Chrome 扩展处理多标签页

#### 技术要求
- Node.js: ^20.19.0 || ^22.13.0 || >=24
- 现代浏览器
- LLM API Key

#### 集成方式
- **CDN 快速体验**：`<script src="https://registry.npmmirror.com/page-agent/1.5.7/files/dist/iife/page-agent.demo.js"></script>`
- **NPM 安装**：`npm install page-agent`

### 示例代码
```javascript
import { PageAgent } from 'page-agent'

const agent = new PageAgent({
  model: 'qwen-plus',
  baseURL: 'https://dashscope.aliyuncs.com/compatible-mode/v1',
  apiKey: process.env.DASHSCOPE_API_KEY,
  language: 'zh-CN',
})

await agent.execute('点击登录按钮')
```
