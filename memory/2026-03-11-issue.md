## 可能的原因

### 1. React Router 路由问题 ❓
- `ProtectedRoute` 检查认证状态
- 如果未登录，重定向到 `/login`
- 可能重定向循环或失败

### 2. JavaScript 运行时错误 ❓
- 编译后的 JS 有两处 `window.location.href =`
- 可能是第三方库（如 axios）的行为
- 可能导致页面白屏

### 3. 认证流程问题 ❓
- `AuthContext` 在加载时显示 "加载中..."
- 如果 API 调用失败，可能卡在加载状态
- token 验证失败但未正确处理

### 4. 依赖库问题 ❓
- Ant Design 可能在 `/admin/` 路径下有问题
- 其他第三方库可能有路径依赖
