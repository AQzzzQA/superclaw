# 股票分析工具使用说明

## 📊 解决方案

已创建本地股票分析工具，**无需网络请求**，完全规避 web_fetch 禁止问题。

---

## 📁 文件说明

```
/root/.openclaw/workspace/
├── stock_data.json      # 股票数据存储（JSON 格式）
├── stock_cli.py        # 命令行工具（主要使用）
└── update_prices.sh   # 批量更新脚本（辅助）
```

---

## 🚀 快速开始

### 方法 1：命令行更新（推荐）

#### 更新单只股票价格
```bash
# 语法
python3 /root/.openclaw/workspace/stock_cli.py update <代码> <价格>

# 示例
python3 /root/.openclaw/workspace/stock_cli.py update 002156 60.50
python3 /root/.openclaw/workspace/stock_cli.py update 002837 105.30
python3 /root/.openclaw/workspace/stock_cli.py update 002895 45.00
python3 /root/.openclaw/workspace/stock_cli.py update 300408 62.80
```

#### 批量更新价格
```bash
# 创建临时文件
cat > /tmp/prices.txt << EOF
002156 60.50
002837 105.30
002895 45.00
300408 62.80
EOF

# 批量更新
cat /tmp/prices.txt | python3 /root/.openclaw/workspace/stock_cli.py batch
```

#### 生成分析报告
```bash
python3 /root/.openclaw/workspace/stock_cli.py report
```

## 💡 使用建议

### 推荐流程

1. **每日收盘后更新价格**
   ```bash
   # 每天收盘后，更新所有股票价格
   cat > /tmp/prices.txt << EOF
   002156 [今日收盘价]
   002837 [今日收盘价]
   002895 [今日收盘价]
   300408 [今日收盘价]
   EOF
   
   cat /tmp/prices.txt | python3 /root/.openclaw/workspace/stock_cli.py batch
   ```

2. **生成分析报告**
   ```bash
   python3 /root/.openclaw/workspace/stock_cli.py report
   ```

3. **查看操作建议**
   - 整体盈利：考虑适当止盈
   - 整体亏损：分析原因，考虑止损
   - 个股表现：重点关注亏损和盈利股票

## 🔧 高级用法

### 手动计算盈亏（如果不想更新价格）

```bash
# 临时计算单只股票盈亏
# 公式: (当前价 - 买入价) × 股数

# 示例 002156
# 如果当前价是 60 元
# 盈亏 = (60 - 54) × 300 = 1,800 元
# 盈亏率 = (60 - 54) / 54 × 100% = 11.11%
```
