## 🚀 快速使用指南

### 启动服务
```bash
cd /root/.openclaw/workspace/openclaw-permission-config

# 快速启动
./quick-start.sh

# 或手动启动
npm start
```

### 访问地址
- **后端API**: http://localhost:8080/api/health
- **前端界面**: http://localhost:3000
- **API文档**: http://localhost:8080/api/docs

### 管理命令
```bash
# 启动服务
./start.sh

# 停止服务
./stop.sh

# 重启服务
./restart.sh

# 查看日志
tail -f logs/server.log
```

- **操作权限验证**: 所有操作都经过权限检查
- **关键操作确认**: 重要操作需要二次确认
- **权限继承验证**: 确保权限关系的合理性

### 2. 配置安全
