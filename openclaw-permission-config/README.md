# OpenClaw 权限配置可视化工具

## 功能说明

这是一个专门为OpenClaw设计的QQ权限配置可视化工具，让您能够轻松管理QQ用户的权限分配。

## 核心特性

### 🔐 安全设计
- **保留管理员最高权限** - 不会覆盖当前管理员配置
- **配置备份机制** - 自动备份当前配置文件
- **权限验证** - 配置应用前进行完整性验证
- **变更日志** - 记录所有权限变更历史

### 🎯 主要功能

#### 1. QQ用户权限管理
- 可视化分配用户权限级别
- 支持批量导入用户列表
- 权限级别分类：管理员、高级用户、普通用户、只读用户

#### 2. 智能权限模板
- 预设常用权限模板
- 自定义权限模板创建
- 模板导出/导入功能

#### 3. 配置文件生成
- 自动生成符合openclaw.json格式的配置
- 实时预览配置文件
- 配置语法验证

#### 4. Web管理界面
- React + TypeScript + Ant Design
- 响应式设计，支持移动端
- 直观的权限配置界面

### 🚀 快速开始

1. **安装依赖**
```bash
cd openclaw-permission-config
npm install
cd frontend && npm install
```

2. **启动服务**
```bash
# 启动后端服务
npm run dev

# 启动前端服务
cd frontend && npm start
```

3. **访问管理界面**
```
http://localhost:3000
```

### 📝 使用说明

#### 用户权限分配
1. 在用户列表中选择用户
2. 选择合适的权限模板
3. 自定义调整权限（可选）
4. 应用配置更改
5. 确认配置生效

#### 配置管理
1. 导出当前配置
2. 导入新配置（可选）
3. 验证配置完整性
4. 应用到系统
5. 查看变更日志
