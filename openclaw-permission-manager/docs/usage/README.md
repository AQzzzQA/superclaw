# 使用指南

## 系统概述

OpenClaw 权限管理系统是一个专门为QQ用户权限配置设计的可视化工具。它提供了完整的用户管理、权限模板管理和配置文件管理功能。

## 主要功能

### 1. 用户管理
- 用户注册和登录（通过QQ号）
- 用户角色分配（管理员、普通用户、只读用户）
- 用户权限设置
- 用户信息编辑和删除

### 2. 权限模板管理
- 创建和管理权限模板
- 系统预设模板
- 自定义模板
- 模板复制和编辑

### 3. 配置管理
- 配置文件管理
- 用户配置生成
- 配置文件导出
- JSON格式预览

### 4. 数据统计
- 用户统计信息
- 权限分布统计
- 系统状态监控

## 用户角色说明

### 管理员 (admin)
- 拥有所有系统权限
- 可以管理用户和权限
- 可以创建和修改权限模板
- 可以管理配置文件

### 普通用户 (user)
- 基础功能权限
- 可以使用大部分系统功能
- 权限有限制

### 只读用户 (readonly)
- 只能查看系统信息
- 不能进行任何修改操作
- 权限受限

## 权限说明

### 基础权限
- **read**: 读取权限，可以查看数据
- **write**: 写入权限，可以创建和修改数据
- **delete**: 删除权限，可以删除数据

### 管理权限
- **manage_users**: 管理用户，可以添加、编辑、删除用户
- **manage_permissions**: 管理权限，可以设置用户权限
- **system_config**: 系统配置，可以修改系统设置

### 消息权限
- **message_send**: 发送消息权限
- **message_view**: 查看消息权限

## 使用流程

### 1. 用户登录

1. 打开系统首页
2. 输入QQ号码
3. 系统自动注册或登录
4. 进入仪表板

### 2. 用户管理

#### 添加用户
1. 点击"用户管理"
2. 点击"添加用户"按钮
3. 填写用户信息：
   - QQ号码（必填）
   - 昵称（必填）
   - 角色选择
   - 权限设置
4. 点击"创建"

#### 编辑用户
1. 在用户列表中找到目标用户
2. 点击"编辑"按钮
3. 修改用户信息
4. 点击"更新"

#### 删除用户
1. 在用户列表中找到目标用户
2. 点击"删除"按钮
3. 确认删除操作

### 3. 权限模板管理

#### 创建权限模板
1. 点击"权限模板管理"
2. 点击"添加模板"按钮
3. 填写模板信息：
   - 模板名称（必填）
   - 描述信息（可选）
   - 权限选择（必填）
4. 点击"创建"

#### 复制权限模板
1. 在模板列表中找到目标模板
2. 点击"复制"按钮
3. 系统自动创建副本
4. 可以修改后保存

### 4. 配置管理

#### 生成用户配置
1. 点击"配置管理"
2. 点击"生成用户配置"按钮
3. 选择目标用户
4. 系统自动生成配置
5. 可以复制或下载配置文件

#### 管理配置文件
1. 在配置列表中查看所有配置
2. 点击"预览"查看配置详情
3. 点击"下载"下载配置文件
4. 点击"编辑"修改配置
5. 点击"删除"删除配置

## 高级功能

### 1. 权限继承

系统支持权限继承机制：
- 管理员角色自动包含所有权限
- 普通用户角色包含基础权限
- 只读用户角色只包含读取权限

### 2. 配置导出

支持多种配置格式：
- JSON格式标准配置
- 美化格式配置
- 用户个性化配置

### 3. 系统监控

提供系统状态监控：
- 用户数量统计
- 权限分布统计
- 系统性能监控

## 最佳实践

### 1. 权限设计原则

- **最小权限原则**：只授予必要的权限
- **角色分离**：不同角色使用不同权限模板
- **定期审查**：定期检查和更新权限设置

### 2. 配置管理建议

- 使用模板化管理权限配置
- 定期备份重要配置
- 使用版本控制管理配置变更

### 3. 用户管理建议

- 根据用户职责分配相应角色
- 定期审查用户权限
- 及时清理不活跃用户

## 常见问题解答

### Q1: 如何修改用户权限？
A1: 在用户管理页面找到目标用户，点击"编辑"按钮，修改权限设置后保存。

### Q2: 权限模板如何应用？
A2: 权限模板创建后，在添加或编辑用户时可以选择相应的权限模板。

### Q3: 如何导出配置文件？
A3: 在配置管理页面，点击目标配置的"下载"按钮即可下载配置文件。

### Q4: 系统支持哪些权限类型？
A4: 系统支持读取、写入、删除、管理用户、管理权限、系统配置、发送消息、查看消息等权限类型。

### Q5: 如何重置用户密码？
A5: 系统支持QQ号自动注册，用户可以通过重新输入QQ号来更新信息。
